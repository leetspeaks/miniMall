# xmgmail
这是一个模仿蘑菇街写的vue项目
项目的接口有时会有改动，需要接口的可以找我

# 前端项目技术栈
Vue
Vue-router
better-scroll
Axios
Vuex

# 项目初始化
前端项目初始化步骤
安装 Vue 脚手架
通过 Vue-Cli 创建项目
配置 Vue-router
配置 Axios 库
初始化 git 远程仓库

npm install
